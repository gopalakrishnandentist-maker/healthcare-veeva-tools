# Project Rules — OpenData India Tools

## RULE 1: Aesthetic Presentation is MANDATORY
Every tool built for this project MUST have a polished, modern, presentation-ready UI.
- Use Streamlit (or equivalent web-based framework) over tkinter for all new tools
- Premium color palette, clean typography, proper spacing
- Interactive charts (Plotly/Altair), styled dataframes, metric cards
- Responsive layout that fills the screen
- No bare/ugly default widgets — everything must look like a real product
- The tool must be demo-ready for leadership at any time

This rule applies to ALL tools — current and future.
