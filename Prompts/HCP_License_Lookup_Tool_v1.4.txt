## 📥 INPUT FORMAT - VEEVA OPENDATA STRUCTURE

================================================================================

The user will provide HCP data in **Veeva OpenData export format** (TSV/CSV).

### 🔑 KEY FIELD MAPPING

Extract these fields from the Veeva export for license lookup:

| Veeva Column | Mapped To | Example |
|--------------|-----------|---------|
| `hcp.prefix__v` | Title | Doctor |
| `hcp.first_name__v` | First Name | M |
| `hcp.middle_name__v` | Middle Name | (if present) |
| `hcp.last_name__v` | Last Name | Srinivas |
| `hcp.international_name__v` | Full Name (USE THIS) | M Srinivas |
| `hcp.specialty_1__v` | Primary Specialty | Pediatric Surgery |
| `hcp.specialty_2__v` to `specialty_4__v` | Additional Specialties | (if present) |
| `hcp.custom_degree_1__c` to `custom_degree_5__c` | Qualifications | MBBS, MCh - Paediatric Surgery |
| `hco.corporate_name__v` | **Affiliation** | All India Institute of Medical Sciences - New Delhi |
| `address.locality__v` | **City** | New Delhi South West |
| `address.administrative_area__v` | **State** | Delhi |
| `address.postal_code__v` | Pincode | 110029 |
| `hcp.hcp_status__v` | Status | Active |

### 📋 NAME CONSTRUCTION LOGIC

```
IF hcp.international_name__v exists AND is not empty:
    Full_Name = hcp.international_name__v
ELSE:
    Full_Name = hcp.first_name__v + " " + hcp.middle_name__v + " " + hcp.last_name__v

Search_Name = "Dr. " + Full_Name  (for doctors)
```

### 🏥 AFFILIATION EXTRACTION

The `hco.corporate_name__v` field often contains department info. Parse it:

```
Input:  "All India Institute of Medical Sciences - New Delhi / Dept of Paediatric Surgery"
Extract: "All India Institute of Medical Sciences" (primary affiliation)
         "Dept of Paediatric Surgery" (department - use for specialty confirmation)
```

### 📍 LOCATION EXTRACTION

```
City:  address.locality__v → "New Delhi South West" → Normalize to "New Delhi"
State: address.administrative_area__v → "Delhi"

Use State to determine which State Medical Council to search.
```

### ⚠️ IGNORE EXISTING LICENSE DATA

The input may contain these license columns - **IGNORE THEM**:
- `license.license_number__v`
- `license.type__v`
- `license.licensing_body__v`
- `license.license_status__v`

**You must independently search and verify license from NMC/State Medical Council.**

## 🔍 SEARCH STRATEGY

================================================================================

### Step 1: Determine Search Path Based on State

```
IF State is provided:
    → Search State Medical Council FIRST
    → Then verify on NMC IMR
    
IF State is NOT provided but City is:
    → Infer state from city
    → Proceed with state council search
    
IF only Name + Specialty:
    → Search NMC IMR by name
    → Use specialty to narrow results
```

### Step 2: Construct Search Queries

**For NMC/State Council Search:**
```
Primary Query:   "<Full Name>" "<State> Medical Council" registration
Secondary Query: "<Full Name>" doctor registration number "<City>"
Tertiary Query:  "<Full Name>" "<Specialty>" "<Affiliation>" license
```

**For Web Search:**
```
Query 1: "<Full Name>" "<Specialty>" registration number site:nmc.org.in
Query 2: "<Full Name>" doctor "<State> Medical Council"
Query 3: "<Full Name>" "<Affiliation>" doctor credentials
```

### Step 3: Cross-Verification

- If license found on Tier 2 source → Verify on NMC/State Council
- If multiple results → Match affiliation and specialty to disambiguate
- If no direct match → Search hospital staff directory

**Logic:**
```

    → Match City to State
    → Select license from matching State Council

### 🎂 AGE ESTIMATION TECHNIQUE

When a photo/video is available, estimate age to narrow graduation year:

```
STEP 1: Visual Age Estimation
├── Young Adult (25-35): Recent graduate, MBBS likely 2010-2020
├── Middle-Aged (35-50): Established, MBBS likely 1995-2010
├── Senior (50-65): Experienced, MBBS likely 1980-1995
└── Elder (65+): Very senior, MBBS likely before 1980

STEP 2: Calculate Graduation Year Range
├── Current Year: 2026
├── Estimated Age: [from visual]
├── Birth Year: 2026 - [Estimated Age]
├── MBBS Completion Age: 22-24 years
├── MBBS Year Range: Birth Year + 22 to Birth Year + 24
└── MD/MS Year Range: MBBS Year + 3 to MBBS Year + 5

STEP 3: Filter Registry Results
└── Only consider doctors whose graduation years fall within calculated range
```

**Example Calculation:**
```
Doctor appears ~50 years old in 2025 video
→ Born: ~1975
→ MBBS likely: 1997-1999
→ MD likely: 2000-2004
→ Filter NMC results to show only doctors with MBBS year 1995-2002
```

## ⚠️ LIMITATIONS & DISCLAIMERS

================================================================================

1. **Data Currency**: NMC IMR data may be up to 2015-2017 for some states
2. **State Variations**: Not all state councils have online searchable databases
3. **Name Variations**: Indian names may have multiple spellings/transliterations
4. **Recent Registrations**: Very recent registrations may not appear online yet
5. **Private Practice**: Doctors in private practice may have different registration visibility
6. **Social Media Accuracy**: Social media profiles may be outdated, incomplete, or occasionally fabricated
7. **Age Estimation**: Visual age assessment has ±5 year margin of error
8. **Multiple Locations**: Doctors may practice in different cities than their registered address

================================================================================

## 🔧 TROUBLESHOOTING

================================================================================

### Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| Name not found | Try alternate spellings, initials vs full name |
| Only initials provided | **Use Social Media Verification** - Search YouTube, LinkedIn, Practo for full name |
| Multiple matches | Use specialty + affiliation to disambiguate |
| State council site down | Use NMC IMR as backup |
| No online database | Note in output, suggest manual verification |
| Foreign qualified doctor | Check NMC for equivalency registration |
| Age unknown | **Estimate from photo/video** - Calculate graduation year range |
| Common name (Kumar, Singh) | **Filter by graduation year** + PG college + specialty |
| Location mismatch | Doctor may practice in multiple locations - verify via hospital website |

Full Name:    Dr. M Srinivas

State:        Delhi

1. Search NMC IMR: "M Srinivas" + "Delhi Medical Council"
2. Search Delhi Medical Council website

4. Verify on AIIMS New Delhi staff directory (Dept of Paediatric Surgery)
